Concepts

Positional numeral systems

Each digit has a place value. For a base b, an integer part is a sum of digits × powers of b: V = Σ dᵢ bᶦ. Fractional parts extend this with negative powers.

  • Binary (base 2): digits 0–1
  • Octal (base 8): digits 0–7
  • Decimal (base 10): digits 0–9
  • Hex (base 16): digits 0–9, A–F
